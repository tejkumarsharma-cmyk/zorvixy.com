# Site Setup Checklist

Use this checklist whenever you create a new site from the template.

## Identity
- [ ] Set `NEXT_PUBLIC_SITE_CODE`
- [ ] Set `NEXT_PUBLIC_SITE_NAME`
- [ ] Set `NEXT_PUBLIC_SITE_TAGLINE`
- [ ] Set `NEXT_PUBLIC_SITE_DESCRIPTION`
- [ ] Set `NEXT_PUBLIC_SITE_DOMAIN`
- [ ] Set `NEXT_PUBLIC_SITE_URL`
- [ ] Set `NEXT_PUBLIC_SITE_OG_IMAGE`

## Theme and Brand
- [ ] Replace logo and favicon
- [ ] Replace default OG image
- [ ] Update theme tokens and fonts
- [ ] Update homepage hero copy
- [ ] Update footer and contact details

## Master Panel
- [ ] Create site record in Master Site Panel
- [ ] Set frontend URL and site code
- [ ] Confirm supported tasks
- [ ] Confirm runtime SEO defaults
- [ ] Confirm sitemap and indexing tracking

## Deploy
- [ ] Add GitHub Actions secrets
- [ ] Add VPS app directory
- [ ] Add DNS records
- [ ] Add nginx host config
- [ ] Enable SSL
- [ ] Confirm GitHub Actions deploy succeeds

## QA
- [ ] Home page loads
- [ ] Search works
- [ ] Feed loads from Master Site Panel
- [ ] Sitemap works
- [ ] Robots works
- [ ] Canonicals are correct
- [ ] Metadata is unique
- [ ] Publish a test post and verify listing/article detail page
